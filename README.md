# Nova Break

This is a Chrome extension that schedules a growing blackhole overlay after a configurable delay.

## Features

- Break timer adjustable from 10 seconds to 10 minutes
- Blackhole overlay that grows and blocks the page
- Manual stop by clicking the overlay or using the popup
- Auto-clear on 30 seconds of Chrome idle time or when the screen locks

## Load unpacked

1. Open Chrome and go to `chrome://extensions`
2. Turn on Developer mode
3. Click Load unpacked
4. Select this folder
