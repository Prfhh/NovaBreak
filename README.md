# 🌌 Nova Break

**A gentle reminder to take breaks—without interrupting your flow.**

Nova Break is a lightweight browser extension that helps you build healthier work habits by displaying a beautiful black hole-inspired visual effect over your current tab when it's time for a break.

Unlike traditional Pomodoro timers that interrupt your work with pop-ups, sounds, or blocked screens, Nova Break stays out of the way. The animation grows gradually in the background while you continue working, typing, scrolling, and clicking as normal.

## 🚀 Installation

Nova Break is currently available as a developer build and can be installed in less than a minute.

### Step 1: Download

Download the `blackhole-break-v1.0.0.zip` file from the latest GitHub release.

### Step 2: Extract

Unzip the downloaded file to any folder on your computer.

### Step 3: Open Extensions

In Chrome or Brave, go to:

```text
chrome://extensions/
```

### Step 4: Enable Developer Mode

Turn on **Developer mode** using the toggle in the top-right corner.

### Step 5: Load the Extension

Click **Load unpacked** and select the extracted Nova Break folder.

That's it! Nova Break is now ready to use.

## ✨ Features

### 🧘 Stay Focused

Get visual break reminders without disruptive notifications or screen takeovers.

### Keep Working Normally

The overlay doesn't block clicks, typing, scrolling, or interactions with the page underneath.

### ⚡ Lightweight Performance

Designed to run smoothly with minimal impact on browser performance.

### Unique Visual Experience

Features a cinematic black hole-inspired animation that gradually appears when a break cycle begins.

### ⚙️ Customizable Intervals

Set work sessions that match your personal workflow and productivity style.

---

## 🔒 Permissions Explained

Nova Break only requests the permissions it needs to function.

### `storage`

Used to save your settings and work interval preferences so they remain available after restarting your browser.

### `activeTab`

Used to display the visual break reminder on the tab you're currently using when a break cycle starts.

---

## 💡 Why Nova Break?

Most break timers demand your attention. Nova Break takes a different approach.

Instead of interrupting your workflow, it gently encourages healthier habits through a subtle visual cue. You stay in control, keep working if needed, and take breaks when you're ready.

A small reminder. A smoother workflow. Better balance throughout your day. 🌌
